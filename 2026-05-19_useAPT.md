# APTコマンド

```terminal
# 確認プロンプトをすべてスキップ
sudo apt purge -y パッケージ名
$ sudo apt purge -y PACKAGE_NAME
# 削除した場合の影響を調べる
$ sudo apt purge --dry-run PACKAGE_NAME
# 削除時に孤立した依存関係を一括削除
$ sudo apt purge --autoremove PACKAGE_NAME
```

## エイリアス

```terminal
# パッケージ最新状態に更新し、不要になったゴミファイルを完全に掃除する
## sysup='sudo apt update && sudo apt upgrade -y && sudo apt autoremove --purge -y
$ sysup
```

## インストールパッケージリスト

```terminal
# Flatpakパッケージを一覧表示
$ flatpak list
# Snapパッケージを一覧表示
$ snap list
# 依存関係を含めたパッケージを一覧表
$ apt list --installed
```
