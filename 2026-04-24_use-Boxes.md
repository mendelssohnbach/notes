## Boxes

# グループに属しているか確認

```terminal
$ groups | grep -E 'kvm|libvirt'
... libvirt kvm  # 2つも必要

# ファイルの所有権と権限を確認
# 実行後、以下のような結果が表示さる。
$ s -l /dev/kvm
crw-rw----+ 1 root kvm 10, 232 4月 24 12:15 /dev/kvm

# KVM 実行権限をユーザーに付与
$ sudo adduser $USER kvm  # ログアウトして再ログイン
```

## ライブラリのインストール

```
# 必要なライブラリが足りていない場合
$ sudo apt update
# sudo apt install qemu-kvm libvirt-daemon-system libvirt-clients bridge-utils -y
```

## 動作確認

```
# 動作確認コマンド
$ vm-ok
INFO: /dev/kvm exists
KVM acceleration can be used
```

## CPU情報の確認

```terminal
# 仮想化が有効かの確認
$ lscpu | grep "仮想化"
仮想化: VT-x

# cpuinfoラグの確認
$ grep -Eoc '(vmx|svm)' /proc/cpuinfo
8 # 1以上の数値
```
