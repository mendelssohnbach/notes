## スコープ

| スコープ           | オプション        | 保存場所         | 用途                       |
| :----------------- | :---------------- | :--------------- | :------------------------- |
| ユーザーグローバル | `--scope user`    | `~/.claude.json` | 全プロジェクトで常に用いる |
| ローカル           | `--scope local`   | `./claude.json`  | プロジェクト単位で用いる   |
| プロジェクト共有   | `--scope project` | `.mcp.json`      | チーム全員で用いる         |

## Context7

```terminal
# プロジェクト単位にインストール
# 現在のディレクトリの .mcp.json に書き込まれます。
// -s は --scope の短縮オプション
$ claude mcp add context7 -s project -- npx -y context7@latest
```

## Playwright

```terminal
# プロジェクト単位にインストール
# 現在のディレクトリの .mcp.json に書き込まれます。
$ claude mcp add playwright -s project -- npx -y @modelcontextprotocol/server-playwright
```

```ClaudeCodeCLI
> mcp list: playwright がリストに含まれているか確認
>mcp status: playwright が Connected になっているか確認
```

## GitHub

```terminal
$ claude mcp add github -scope project -- npx -y @modelcontextprotocol/server-github
```

| できること     | 具体的な操作例                                         |
| :------------- | :----------------------------------------------------- |
| Issue管理      | 「このIssueの内容をもとにコードを修正して」            |
| PRレビュー     | 「開いているPRの変更点を確認して問題があれば指摘して」 |
| コード横断検索 | 「リポジトリ全体でこの関数が使われている箇所を探して」 |
| ブランチ操作   | 「featureブランチを切って変更をコミットして」          |

## PostgreSQL

```terminal
$ claude mcp add postgres --scope project -- npx -y @modelcontextprotocol/server-postgres
```

| できること           | 説明                                   |
| :------------------- | :------------------------------------- |
| 自然言語クエリ       | SQLを知らなくても操作できる            |
| スキーマ確認         | 「このテーブルの構造を教えて」が使える |
| マイグレーション支援 | 変更内容を確認しながら安全に作業できる |
