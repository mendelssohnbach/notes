## Context7

```terminal
# プロジェクト単位にインストール
# 現在のディレクトリの .mcp.json に書き込まれます。
$ claude mcp add context7 -s project -- npx -y context7@latest
```

## Playwright

```terminal
# プロジェクト単位にインストール
# 現在のディレクトリの .mcp.json に書き込まれます。
$ claude mcp add playwright -s project -- npx -y @modelcontextprotocol/server-playwright
```

```ClaudeCodeCLI
> mcp list: playwright がリストに含まれているか確認
>mcp status: playwright が Connected になっているか確認
```
