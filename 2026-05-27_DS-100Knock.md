## 学びのまとめ

正規表現を用いる

`~` を用いる。文字列をシングルクォーテーション `'...'` で囲む

```sql
SELECT
    *
FROM
    store
WHERE
    tel_no ~ '[0-9]{3}-[0-9]{3}-[0-9]{4}'
LIMIT 10;
```

ウィンド関数

`RANK()` と `OVER()` はセットで用いる。動詞+前置詞の並びである。指定したデータ全体にまたがって、ランク付けすると読み取れる。

```sql
SELECT
    customer_id,
    amount,
    -- RANK()とOVERは1つのウィンド関数のセット
    -- RANK() : 順位をつける関数命令/順位付けされた位が入る
        -- 特定の列の値が入るのではない
    -- OVER() : 範囲やルールで順位をつける構文(対象)
        -- (...)の中はルールそのもの
    RANK() OVER(ORDER BY amount DESC) RANKING
FROM
    receipt
LIMIT 10;
```

`ROW_NUMBER()` 等しい場合でも別順位で順位付けする。

```sql
SELECT
    customer_id,
    amount,
    ROW_NUMBER() OVER(ORDER BY amount DESC) AS RANKING
FROM
    receipt
LIMIT 10;
```

`DISTINCT` 値の重複なくす/ユニークな値のみにする。

```sql
SELECT
    COUNT(DISTINCT customer_id)
FROM
    receipt;
```

`SUM()` 合計する。

```sql
SELECT
    store_cd,
    SUM(amount) AS sum_amount,
    SUM(quantity) AS sum_quantity
FROM
    receipt
GROUP BY
    store_cd
LIMIT 10
;
```

`MAX()` 最も新しい。DBでは並び順を保証しない。最も新しい=最も未来=最も数値が大きい

```sql
SELECT
    customer_id,
    MAX(sales_ymd) AS LATEST_SALES_YMD
FROM
    receipt
GROUP BY
    customer_id
LIMIT 10
;
```

`MIN()` 最も古い。DBでは並び順を保証しない。最も古い=最も過去=最も数値が小さい

```sql
%%sql
SELECT
    customer_id,
    -- 小文字を積極手に用いる
    -- 内部で小文字に変換さえる
    MIN(sales_ymd) AS earliest_sales_ymd
FROM
    receipt
GROUP BY
    customer_id
LIMIT 10
;
```

`avg()` 平均を求める

```sql
select
    store_cd,
    avg(amount) as avg_amount
from
    receipt
group by store_cd
order by avg_amount desc
limit 5;
```

`percentile_cont()` 指定された割合に位置する値を求める。 `0.5` を指定すれば中央値が求まる。 `within group` とセットで用いる。 `within group` にてどの列をどのように並べるかを指定する。

```sql
select
    store_cd,
percentile_cont(0.5) within group(order by amount) as amount_median
from
    receipt
group by store_cd
order by amount_median
limit 5
;
```

`var_pop()` 分散を求める

```sql
select
    store_cd,
    var_pop(amount) as vars_amount
from
    receipt
group by store_cd
order by vars_amount desc
limit 5;
```

`stddev_pop` 標準偏差を求める

```sql
select
    store_cd,
    stddev_pop(amount) as stds_amount
from receipt
group by store_cd
order by stds_amount desc
limit 5;
```

内部結合 table_name_a a join table_name_b b on a.column_x = b.column_x

```sql
select
    r.*,
    s.store_name
from receipt r
join store s
on
    r.store_cd = s.store_cd
limit 10;
```

`coalesce` 引数として渡されたリストの中から、最初に `null` でない値を返す

```sql
    coalesce(a.sum_amount, 0)
```

`lag()` 現在の行よりも前の行の値を取得する。第2引数に何行前なのかを指定し、第3引数で前の行が存在しない場合に返す値

```sql
   lag(sales_ymd, 1) OVER(ORDER BY sales_ymd) lag_ymd,
   lag(amount, 1) OVER(ORDER BY sales_ymd) AS lag_amount
```
