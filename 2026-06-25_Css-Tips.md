## with

`with` : `min(Npx, calc(...))`
: PC時は560px固定、スマホ時は画面幅に追従

```css
width: min(560px, calc(100% - 32px));
```

## clamp

`clamp` : (最小値, 推奨値, 最大値)
: 画面の幅（ビューポートの幅）に合わせて文字の大きさを「40px から 80px の間」で自動的に変化させる

```css
font-size: clamp(40px, 8vw, 80px);
```

## grid-template-columns

`grid-template-columns` : `1col 2col`
要素を2列に2列に並べる

- 1列目: auto / 中身のコンテンツに合わせて幅を決定
- 2列目: 1fr / 残りのスペースをすべて専有

```css
grid-template-columns: auto 1fr;
```

`grid-template-columns` : `repeat(...);`
グリッドの列の数と幅を指定

- `repeat` : (数, 幅)繰り返しの指定
- `repeat(3, ...)` : 同じ幅を3列作成
- `minmax(0, 1fr)` : 列の最小幅 0 と
- `1fr`：最大幅/空き領域をを均等に「1つ分の割合」として分ける

```css
grid-template-columns: repeat(3, minmax(0, 1fr));
```

## 子孫セレクタ

`.search-box` クラスの内側の `span` 要素が対象

```css
.search-box span {
  ...
}
```

`.search-box` クラスの内側の `input` 要素が対象

```css
.search-box input {
  ...
}
```

` input` 要素に `:focus` 疑似クラスを付加する

```css
.search-box input:focus {
  ...
}
```

## 属性セレクター

**属性セレクター**
: 特定のHTML属性を持つ要素やHTML属性に特定の値を持つ要素。 `{}` で囲む

```html
[data-type='primary'] { ... }

<div data-type="primary"></div>
```

何らかの属性を持つ要素

```html
[data-type] { ... }

<div data-type="primary"></div>
<div data-type="secondary"></div>
```

```css
[data-type='primary' s] {...}
```

```css
/* https から始まらない（http などのリンク） */
a:not([href^='https']) { ... }
```

| 演算子  | 機能                     |
| :------ | :----------------------- |
| `s`     | 大文字小文字を区別する   |
| `i`     | 大文字小文字を区別しない |
| `*=`    | 部分一致                 |
| `^=`    | 前方一致                 |
| `$=`    | 前方一致                 |
| `=`     | 完全一致                 |
| `a:not` | 各種一致を否定           |
