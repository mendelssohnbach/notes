```terminal
# pdfseparate
$ sudo apt install pdftk -y

# 全ページを1ページずつ分割
$ pdftk input.pdf burst

# 1ページ目と3~5ページ目だけを抽出
$ pdftk input.pdf cat 1 3-5 output selected_pages.pdf
```

## ワンライナー

カレントディレクトリに `pagination.csv` に作成する。

```csv
2,5
6,15
75,106
144,172
```

```terminal
$ awk -F',' '{printf "$ pdftk input.pdf cat %d-%d output selected_pages%03d-%03d.pdf\n", $1, $2, $1, $2}' pagination.csv > separate.txt
```

- awk -F',': カンマを区切り文字として指定する。
- printf: 指定したフォーマットで文字列を生成する。
- %d: 整数を表示する。
- %03d: 整数を 3桁 で表示し、足りない場合は 0で埋める（例: 2 → 002）。
- $1, $2: CSVの1列目と2列目の値を参照する。
- `> separate.txt`: 実行結果をファイルに保存する。
