## エイリアス

```txt
cp = cherry-pick
tree = log --graph --pretty=format:'%x09%C(auto) %h %Cgreen %ar %Creset%x09by"%C(cyan ul)%an%Creset" %x09%C(auto)%s %d'
# ステージングされたすべての変更を取り消す
unstage = !git restore --staged $1
# Gitの管理から永久に除外
outcast = git rm --cached
# 直近のコミットログを表示
last = log -1 HEAD
# 現在のコミットのハッシュを取得
current-hash = show --format='%h' --no-patch
# 最後のn個のコミットの歴史を修正操作
fix = !sh -c 'git rebase -i HEAD~${1}'
# 現在の状態を簡潔な形式で表示
st = status -s
ci = commit
# 変更をすべてステージング領域に追加
aa = add -A
# 追跡されていないファイルとディレクトリを削除
cdf = clean -df
# 直前のコミットに追加し、新しいコミットメッセージの入力をスキップ
amend = commit -a --amend --no-edit
# 空白の違いを無視する
dw = diff -w
# 最新の10件の結果を表示
hist = log --oneline -10
# 新規ファイルをdiff表示できるように追加
born = add --intent-to-add
# 詳細なコミットメッセージをグラフ形式で表示
body = log --pretty=format:\"%h: %B\" --graph
# リモートリポジトリから取得し差分を表示
probe = fetch && git diff main origin/main
```

## コマンド

コミットメッセージのままでプルリクエストを即座に作成

```terminal
$ gh pr create --fill
```

ローカルブランチの一覧と、リモートブランチの紐付け状態を詳しく表示

```terminal
$ git branch -vv
```
