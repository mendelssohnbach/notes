**awk** は、大きな生データからレポートを生成するような作業に便利である。

## 構文

```terminal
awk [OPTION] '{ PATTERN }' FILE_NAME
```

## 組み込み変数

NR
:　行番号を表示(Number of Record)
NF
: 行の列数を表示(Number of Field)
FS
: 区切り文字を指定(Field Separator)
OFS
: データ主力時の区切り文字を指定(Output Field Separator)

## シェバン

```awk
#!/usr/bin/env -S awk -f
```

- S: `env` コマンド以降の文字列をスペース区切りで、個別の引数として渡せる

## リンター

```terminal
$ awk -f SCRIPT_NAME /dev/null

awk: SCRIPT_NAME: line 2: syntax error at or near [
awk: SCRIPT_NAME: line 2: extra '}'
```

## ワンライナー

ファイルの行数を表示

```terminal
$ awk 'END {print NR}' inventory-shipped
17
```

指定した行の表示

````terminal
$ awk 'NR == 1' bbs-list
ardvark     555-5553     1200/300          B

$ awk 'NR == 3' bbs-list
barfly       555-7685     1200/300          A
```

偶数行/奇数行の表示

```terminal
# ファイルの偶数行を表示
$ awk 'NR % 2 == 0' bbs-list
alpo-net     555-3412     2400/1200/300     A
bites        555-1675     2400/1200/300     A
core         555-2912     1200/300          C
foot         555-6699     1200/300          B
sdace        555-3430     2400/1200/300

# ファイルの奇数行を表示
$ awk 'NR % 2 == 1' bbs-list
ardvark     555-5553     1200/300          B
````
