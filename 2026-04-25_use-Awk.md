**awk** は、大きな生データからレポートを生成するような作業に便利である。

## 構文

```terminal
awk [OPTION] '{ PATTERN }' FILE_NAME
```

## 組み込み変数

NR
:　行番号を表示(Number of Record)
NF
: 行の列数を表示(Number of Field)
FS
: 区切り文字を指定(Field Separator)
OFS
: データ主力時の区切り文字を指定(Output Field Separator)

## シェバン

```awk
#!/usr/bin/env -S awk -f
```

- S: `env` コマンド以降の文字列をスペース区切りで、個別の引数として渡せる

## リンター

```terminal
$ awk -f SCRIPT_NAME /dev/null
```

## 入力ファイルの読み込み

レコード(行)単位で1レコード毎に読み込まれ処理される

FILENAME
: 入力ファイルのファイル名変数
RS
: 改行変数。デフォルトは改行コードだが指定も可能。引用符で囲むこと。

```terminal
# / で改行。/が改行に置換されている
$ awk 'BEGIN { RS = "/" } ; { print $0 }' bbs-list
ardvark     555-5553     1200
300          B
alpo-net     555-3412     2400
1200
300     A

# 0 で改行
$ awk 'BEGIN { RS = "0" } ; { print $0 }' bbs-list
ardvark     555-5553     12

/3

          B
alpo-net     555-3412     24

/12

/3

     A
```

### フィールドの検査

$1
: フィールドを表す
$0
: 1レコードを表す

```terminal
# 第1フィールドが foo で始まる行をすべて出力
$ awk '$1 ~ /foo/ { print $0 }' bbs-list
fooey        555-1234     2400/1200/300     B
foot         555-6699     1200/300          B
macfoo       555-6480     1200/300          A
sabafoo      555-2127     1200/300          C

# レコード全体から foo を探し、
# マッチしたレコードの第1フィールドと最終フィールドを表示
$ awk '/foo/ { print $1, $NF }' bbs-list
fooey B
foot B
macfoo A
sabafoo C
```

### フィールドの内容変更

```terminal
# 算術
# 第3フィールドに第2フィールド値-10の結果を代入
$ 最終フィールドに第2フィールドに第4フィールドを乗算した結果を代入
$ awk '{ $3 = $ 2 - 10; $NF = $2 * $4; print $2, $3, $NF }' inventory-shipped
13 3 195
15 5 360
15 5 510
31 21 1953
```

出力時に算術結果に置き換わる

```terminal
# 出力結果の第2フィールドの値は置き換わる
$ awk '{ $2 = $2 - 10; print $0}' inventory-shipped
Jan 3 25 15 115
Feb 5 32 24 226
Mar 5 24 34 228
Apr 21 52 63 420
```

## 出力

### print

**print** 文は単純で標準的な書式で出力する。

```terminal
# 改行 \n を含む文字列の出力
$ awk 'BEGIN { print "line one\nline two\nline three"}'
line one
line two
line three
```

```
# 2つのフィールドを空白で区切って出力
$ awk '{ print $1, $2}' inventory-shipped
Jan 13
Feb 15
Mar 15
Apr 31
```

```
# 2つのフィールドを結合して出力
$ awk '{ print $1 $2}' inventory-shipped
Jan13
Feb15
Mar15
Apr31
```

### output separator

組み込み変数 **OSF** に任意の文字列をセットすることで出力フィールドセパレータを指定できる。

```terminal
# `,` 区切りで出力
$ awk 'BEGIN { OFS = "," } { print $1, $2 }' bbs-list
ardvark,555-5553
alpo-net,555-3412
barfly,555-7685
```

### printf

複雑な書式で出力する。アイテムごとに幅を指定したり、数値を出力スタイルでを指定できる

```terminal
# 第1フィールドを20文字の文字列とする
# 第2フィールドを文字列とする
$ awk '{ printf "%-20s %s\n", $1, $2 }' bbs-list
ardvark              555-5553
alpo-net             555-3412
barfly               555-7685
bites                555-1675
```

## パターンとアクション

- パターンを省略した場合: 全てのレコードに対してアクションを実行
- アクションを省略した場合: パターンにマッチしたレコードを表示
- パターン/アクション共に省略: 何も表示しない

## 真偽の判定

| 条件式   | 判定 |
| :------- | :--- |
| `0`      | 偽   |
| 空文字   | 偽   |
| 上記以外 | 真   |

```terminal
# アクションがない
# 数値0がパターンの場合
$ echo "foo" | awk '0'  # 何も表示されない
# 数値4がパターンの場合
$ echo "foo" | awk '4'
foo
# 文字列の0がパターンの場合
$ echo "foo" | awk '0""' # 数値の後に "" で文字列扱い
foo
# 文字列がパターンの場合
$ echo "foo" | awk '"bar"'
foo
```

## 比較演算子、マッチング演算子

```terminal
$ echo "foo" | awk '$0 == "foo"'
foo
# 真の場合 1 が返される
$ echo "foo" | awk '{ print $0 == "foo" }'
1
# 偽の場合 0 が返される
$ echo "foo" | awk '{ print $0 != "foo" }'
0
```

正規表現は `~ /STRING/` と記述する。

```terminal
# マッチング演算子
$ echo "foo" | awk '$0 ~ /foo/'
foo
# 正規表現に限り `$0 ~` を省略できる
$ echo "foo" | awk '/foo/'
foo

# 真の場合 1 が返される
$ echo "foo" | awk '{print $0 ~ /foo/}'
$ echo "foo" | awk '{print /foo/}' # 省略表記
1
# 偽の場合 0 が返される
$ echo "foo" | awk '{print $0 ~ !/foo/}'
$ echo "foo" | awk '{print !/foo/}' # 省略表記
0
```

## 代入演算子

破壊的代入が可能である。

```terminal
$ echo "a b c" | awk '{$2 = "B"; print $0}'
a B c
# 文字列は " でクォートする
$ echo "a b c" | awk '{$1 = "えー"; $2 = "β"; $3 = "C"; print $0}'
えー β C
```

削除の場合は、代入の戻り値は左辺値である。

```terminal
# 削除
$ echo "a b c" | awk '{$2 = ""; print $0}'
a  c
# 代入の戻り値が代入の真偽でない場合
$ echo "a b c" | awk '$2 = ""' # 何も返されない
# 代入値を確認
$ echo "a b c" | awk '{print $2 = "B"}'
B
$ echo "a b c" | awk '{print $2 = ""}' # 何も返されない
$ echo "a b c" | awk '!($2 = "")'
a  c
```

## 変数

| 組み込み変数 | 機能         | フルネーム       |
| :----------- | :----------- | :--------------- |
| `NR`         | レコード数   | Number of Record |
| `NF`         | フィールド数 | Number of Field  |

```terminal
$ echo "a b c" | awk 'NF == 3'
a b c
$ echo "\n a \n b \n \n" | awk 'NF'
\n a \n b \n \n # 改行が認識されない
$ echo -e "\n a \n b \n \n" | awk 'NF' # -e オプションが必要
 a
 b
$ echo -e "¥n a ¥n b ¥n ¥n" | awk 'NF'
```
