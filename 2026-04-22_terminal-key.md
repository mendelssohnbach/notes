コマンドラインショートカット

## 移動

| キー   | 機能                 |
| :----- | :------------------- |
| Ctrl+B | 前方の単語頭に移動   |
| Alt+B  | 1文字前に移動        |
| Alt+L  | 後方の単語終端に移動 |
| Alt+b  | 1単語分左に移動      |
| Alt+f  | 1単語分右に移動      |
| Esc>b  | 1単語分前に移動      |
| Esc>f  | 1単語分後に移動      |

## 削除

| キー | 機能 |
| Ctrl+W | カーソルの前にある1単語を削除 |
| Ctrl+U | カーソル位置から行頭まで全削除 |
| Ctrl+K | カーソル位置から末尾まで全削除 |
| Ctrl+H | カーソルの前1文字を削除 |
| Ctrl+D | カーソル上の1文字を削除 |
| Ctrl+Y | 削除した文字の貼り付け |
